Bienvenido a SPA Ceiba

SPA Ceiba es una iniciativa de Edwin Gamboa para el ADN Ceiba, con el fin de fortalecer los conocimientos obtenidos.

Descripción:

  • Estas citas solo pueden ser tomadas de Lunes a Sábado.
  • Estas citas no pueden ser tomadas para el dia actual
  • Estas reservas pueden ser para cualquier producto registrado
  • Cada producto registrado cuenta con su propio valor
  • Cada producto puede o no generar un cupón para futuras compras
  • El porcentaje de descuento del cupón a generar puede variar para cada producto
  • En las compras en que se usa un cupón de descuento no generará un nuevo cupón

    • Para dar solución a este problema se hizo uso de un Back en Java y el Front en Angular.
